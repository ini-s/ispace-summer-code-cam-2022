About

iSPACE is an exciting and fun tech club that helps Children and Teens understand the world of gaming, coding, programming, mechatronics, IOT, music programming and computer electronics.
The acronym iSPACE means Introduction to Systematic Programming, Animation, Computer Engineering and Electronics.
iSPACE is a co-curricular academic program designed to train students in a career pathway and also develop innovative and creative abilities / skills that will be useful in other areas of learning and problem solving.

Courses

Robotics

Robotics is the branch of science focused on learning about and creating robots or machines that can do work. You will learn how to bring robots to life with code, work with sensors and microcontrollers.

Python

Python is a computer programming language. It lets kids turn their ideas into a tangible on-screen result. They can create their own games, programs, animation problem solving exercises, websites, and even integrate with robotics.

Animation

Animation is the process of giving the illusion of movement to drawings, models, or inanimate objects. You will learn how to create animated videos and stories.

Graphic Design

You will learn the principles and elements of graphic design using photoshop.

3D Game Development

You will learn how to create worlds, mod your favorite titles, design 3D levels, and learn the cycle of game production.

Mobile Application Development

Using App Inventor which is a free, cloud-based service that allows you to make your own mobile apps using a block based programming language. You will learn the basics of programming apps for Android devices.

Web Development

In this course, you will learn how to build a webpage, with HTML style it with CSS and add a key concept to all webpages, make it mobile responsive.

Artificial Intelligence

This Course introduces machine learning by providing hands-on experiences for training machine learning systems and building things with them.

3D Modelling

In this you will get an in-depth look at the first steps necessary to building high-resolution characters for video games or movies.
